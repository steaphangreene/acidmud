AcidMUD: Quickstart Guide

Quickstart/FAQ:

  • See Connect Info to just play with the running instance.
  • Create one account for you (the player) not one-per-character.
  • Create a character in the default (TBA) world with: 
    newch [Name]
  • Set up character using reasonable defaults with: 
    arch Fighter
  • Tweak things if you want, to see options use: 
    help commands
  • Start using character with: 
    enter [Name]
  • Before attacking anything, first see if that's a good idea with: 
    consider [target]
  • Characters don't respawn when they die. Make a new character.

Note that AcidMUD is just a system, it does not have any content, so it has been configured to load and roughly translate some tbaMUD content instead. I make no claim to ownership or authorship of any of that content. I am using it with permission, as far as I know. If someone knows that is not the case, please let me know right away.
